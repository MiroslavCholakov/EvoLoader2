# EvoLoader2

An updated version of the Ziadoma EvoLoader. This program is used to load your hero codes into Warcraft 3.
It can also be used to track the new godly recipies and monitor the guides directly from the app instead of the discord #guides

Now works with all Twilight maps :)



\## Download



Latest version:

\[Download EvoLoaderII v1.3](https://github.com/MiroslavCholakov/EvoLoader2/releases/tag/TeveEvo )]



\### Installation



1\. Download the latest `.rar`

2\. Extract it anywhere ( Preferably not on C:\ if on Windows)

3\. Run `EvoLoaderII.exe`

4\. Select your Warcraft III folder on first launch
